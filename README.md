# IRO-PROJECT (Issues Reporting Outlet)

A web-based Issues Reporting Outlet (IRO) developed as a CSC426 Software Engineering project for Adekunle Ajasin University, Akungba-Akoko (AAUA).

The system provides a centralized platform that enables AAUA students and community indigenes to report community-related issues, monitor the progress of their reports, and allows administrators to review and manage submitted reports efficiently.

---

## Project Objectives

- Provide an online platform for reporting community issues.
- Improve communication between users and administrators.
- Enable efficient management of reported issues.
- Allow users to track the status of submitted reports.
- Promote accountability through a structured reporting system.

---

## System Features

### User Features

- User Registration
- User Login
- Report Community Issues
- View Submitted Reports
- Track Report Status
- Logout

### Administrator Features

- Administrator Login
- Review Submitted Reports
- Update Report Status
- Manage Users
- Logout
